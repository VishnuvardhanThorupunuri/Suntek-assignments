#  Suntek Academy — Week 6 Web Development Assignments

Welcome to the **Week 6 Assignments Directory**! This repository showcase focuses on mastering advanced modern CSS layouts, responsive web design patterns, CSS Grid, Flexbox alignment, and framework-inspired structural patterns.

Each project in this directory is a self-contained layout designed to solve specific challenges in front-end development, ranging from clean editorial landing pages to highly complex testimonial grids.

##  Detailed Assignment Breakdown

---

###  Assignment 9: Travel Layout
A minimalist, visual-heavy travel agency landing page built to master background overlay techniques and clean media-to-text sections.

*   **Key Features:**
    *   **Hero Section:** A full-width background hero image accompanied by a high-contrast dark overlay to ensure dynamic heading text readability.
    *   **Call-to-Action (CTA):** A clean "Discover" button encouraging user interaction.
    *   **Outdoor Activities Area:** A classic two-column section pairing descriptive marketing text with high-resolution imagery.
*   **Aesthetic & Styling highlights:**
    *   Responsive text layouts.
    *   Fluid vertical margins and paddings for readable spacing.
*   **Technologies Used:** HTML5 Semantic Tags, CSS3 (Flexbox/Block-level layouts).

---

###  Assignment 10: Libertarians Layout
A bold, editorial card-based layout featuring political theory quotes and themes, designed to test grid columns and structured spacing.

*   **Key Features:**
    *   **Branded Header Bar:** A simple, high-visibility branding header logo.
    *   **Hero Message:** A large, modern-serif font banner statement ("Down with the state") backed by supporting text and custom logo graphics.
    *   **Core Principle Cards:** A responsive 3-card horizontal grid raising key economic questions, styled with neat borders, modern padding, and soft shadow highlights.
*   **Aesthetic & Styling highlights:**
    *   Card hover states.
    *   Contrasted color scheme prioritizing readability.
*   **Technologies Used:** HTML5 Semantic Structure, Custom CSS3 flex/grid properties.

---

###  Assignment 11: Responsive Testimonial Grid
An advanced grid system modeled after standard testimonial boards (similar to the Frontend Mentor challenge). It demonstrates high-level mastery of **CSS Grid Layout** spanning.

*   **Key Features:**
    *   **Complex Grid Layout:** Testimonials spanning different widths and heights, using direct column/row grid alignments.
    *   **Verified Profiles:** Each card contains custom avatar images, user names, and verified status badges mimicking modern UI/UX card designs.
    *   **Dynamic Visual Styling:** Varied background colors per card, custom gradient text, high-end profile layouts, and polished drop-shadow depth indicators.
*   **Aesthetic & Styling highlights:**
    *   Dynamic media-query responsive flow changing grid shapes across mobile, tablet, and desktop views.
    *   Modern borders, smooth profile transitions, and card rounding.
*   **Technologies Used:** HTML5, CSS Grid (`grid-template-columns`, spans), CSS Flexbox (for profile headers).

---

###  Bootstrap Assignment 1: Jumbotron Marketing Template
A replica of the classic **Bootstrap Jumbotron** layout built using pure custom CSS to mimic layout structures found in standard frameworks.

*   **Key Features:**
    *   **Jumbotron Header:** Large marketing banner offering an informative website template starter.
    *   **Three-Column Marketing Grid:** Standard multi-column row with "View details >>" CTAs underneath descriptive paragraphs.
    *   **Semantic Footer:** Standard copyright branding and page divider.
*   **Aesthetic & Styling highlights:**
    *   Classic clean styling borders and layouts matching standard Bootstrap design system principles.
    *   Well-spaced typography utilizing consistent line-heights.
*   **Technologies Used:** HTML5 elements, Vanilla CSS components.

##  How to Run & Preview

To preview any of the assignments locally:

1.  **Direct Browser Access:**
    *   Navigate into any assignment directory (e.g., `Assignment-11`).
    *   Double-click the `index.html` file to open it directly in your default browser.
2.  **Using VS Code Live Server (Recommended):**
    *   Open the `week-6` directory in VS Code.
    *   Make sure you have the **Live Server** extension installed.
    *   Open `index.html` in your active tab.
    *   Click the **Go Live** button at the bottom-right corner of the editor to run a live hot-reloading development server on `localhost:5500`.

##  Core Skills Demonstrated in Week 6

*   **Advanced CSS Grid:** Spanning items across multiple rows and columns cleanly without breaking flow.
*   **Dynamic Flexbox Alignments:** Creating navigation headers, card bodies, and alignment modules.
*   **Mobile-First Responsive Layouts:** Writing robust `@media` query blocks to restructure grids dynamically between mobile and desktop devices.
*   **Component UI Card Design:** Structuring user avatars, badges, card blocks, and CTA elements with professional look-and-feel design languages.
